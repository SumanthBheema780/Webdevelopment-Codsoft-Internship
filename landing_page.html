Core Knowledge

Everything you'll learn building one page.

A landing page is deceptively rich. Nail it and you've mastered the fundamentals that power every website on the internet.

⬛

Build Process

From blank file to live page.

Follow this sequence. Don't skip steps — each one builds on the last.

01

Plan your layout

Sketch on paper first. Decide on sections: nav, hero, features, testimonials, CTA, footer. Know before you code.

02

Write the HTML

Build the skeleton. Semantic HTML only. Don't style yet — structure comes first, appearance follows.

03

Apply CSS

Start with layout (flexbox/grid), then spacing, then colors and fonts. Work from macro to micro.

04

Polish & refine

Hover states, transitions, font pairing, color refinement. This is where good designs become great ones.

Key Design Elements

Pay attention to these.

⬡
Alignment
Everything should align to an invisible grid. Misalignment is the #1 sign of an amateur design. Use flexbox alignment properties religiously.
□
Boxes & Cards
Learn border, border-radius, box-shadow, and overflow. Cards are the universal container of the web — master them early.
↕
Padding
Content that breathes feels premium. Generous internal spacing separates polished pages from cramped, busy ones.
◑
Color Palette
Choose 3 colors max: one dominant, one accent, one neutral. Use CSS variables so you can tweak the whole palette in one place.
≡
Z-index & Overlap
Understand the stacking context. Overlapping elements look great when intentional — catastrophic when accidental.
